# AtollRPC Wiki

Welcome to the AtollRPC documentation. AtollRPC is a Swift SDK for third-party macOS apps to display **live activities**, **lock screen widgets**, and **notch experiences** in [Atoll](https://github.com/Ebullioscopic/Atoll) via JSON-RPC over WebSocket.
